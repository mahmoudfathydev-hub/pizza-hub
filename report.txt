PROJECT REPORT - PizzaHub
========================================

Project Overview
----------------
PizzaHub is a modern, responsive food ordering web application built with Next.js and React. The project aims to provide a seamless user experience for browsing menus, viewing offers, and placing orders. The design focuses on a premium, appetite-inducing aesthetic with a user-friendly interface.

Work Accomplished
-----------------
1. **Core Infrastructure**:
   - Initialized Next.js 15+ project with TypeScript.
   - Configured Tailwind CSS v4 for styling.
   - Set up Prisma with PostgreSQL for database management.

2. **Components & UI**:
   - **Footer**: Implemented a comprehensive Footer component (`src/components/footer`) including:
     - Newsletter subscription field.
     - Quick navigation links.
     - Contact information (Address, Phone, Email).
     - Operating hours.
     - Social media integration slots.
     - Fully responsive design (stacks on mobile, grid on desktop).
   - **Header/Navbar**: Existing structure for navigation (`src/components/header`).
   - **Menu System**: Components for displaying menu items (`src/components/menu`).
   - **UI Library**: Integrated accessible UI components (Button, Dialog, Checkbox, etc.) in `src/components/ui`.

3. **Styling & Assets**:
   - Implemented a consistent color theme (Primary: Orange, Secondary: Dark/White).
   - Responsive layouts ensuring compatibility across devices.

Code Statistics
---------------
- **Total Files**: ~65+ files in `src` directory.
- **Key Directories**:
  - `src/app`: Application routes and pages.
  - `src/components`: Reusable UI components.
  - `src/lib`: Utility functions and database clients.

Technologies Used
-----------------
- **Framework**: Next.js 16.1.4 (React 19)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4, Lucide React (Icons), `clsx`, `tailwind-merge`
- **Database**: PostgreSQL with Prisma ORM (v7.3.0)
- **UI primitives**: Radix UI (Dialog, Slot, etc.)
- **Animations**: AOS (Animate On Scroll)
